# **Isotopes**

|      | Learning Outcome                                                                       |
| ---- | -------------------------------------------------------------------------------------- |
| 1(e) | (ii)distinguish between isotopes on the basis of different numbers of neutrons present |

Isotopes are atoms of an element that have the **same proton number but different nucleon numbers** (or **same number of protons and different number of neutrons**).

Isotopes of an element have the same electronic configuration and thus have **similar chemical properties** (undergo similar reactions).

They have different relative isotopic masses and thus **different physical properties** (density etc).
